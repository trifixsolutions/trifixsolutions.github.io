## 2025-05-14 - Accessible Form Feedback Pattern
**Learning:** For static sites without complex frameworks, a simple `aria-live` container combined with CSS `:empty { display: none; }` provides an elegant, accessible way to replace jarring browser alerts with in-page feedback.
**Action:** Use the `#form-status` pattern with success colors (`#ecfdf5`, `#065f46`) for consistent micro-UX feedback in similar projects.
